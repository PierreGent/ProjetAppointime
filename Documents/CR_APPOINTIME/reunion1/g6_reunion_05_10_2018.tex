\subsection{Reflexion sur le support de l'application}
Nous avons réfléchi sur la nécessité de faire une application mobile
et un site web plutôt qu'un site web responsive. En effet, une
application mobile serait nécessaire uniquement si nous trouvons des
arguments pour produire des interactions utiles entre le mobile et
l'application : notifications, alarmes, map ... Nous avons mis cette
question en attente jusqu'à la definition des différentes
fonctionnalités.
